# BlackRoad Discord Bot

Community bot for BlackRoad OS Discord server.

## Commands

- `/status` - Check infrastructure status
- `/agents` - List active AI agents
- `/help` - Get help links

## Setup

1. Create bot at discord.com/developers
2. Set `DISCORD_TOKEN` environment variable
3. Run `npm install && npm start`
